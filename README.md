# Hotel_Management_system
C++ Hotel Management System with room booking, service queue, reservations, billing, and discounts.
This is a menu-driven Hotel Management System implemented in C++. It allows users to:
Display available rooms and their details.
Book rooms and manage room availability.
Add or cancel reservations using a stack-based history.
Manage service requests using a queue.
Generate bills recursively including additional charges.
Apply discounts based on stay duration and customer loyalty.
